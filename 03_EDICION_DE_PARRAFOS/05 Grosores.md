# Grosores

## ¿Cuáles son los dos grosores que nos permite latex en las fuentes?

- Grosor normal `\textmd{Text}`

- Grosor negritas `\testbf{Text}`

## ¿Cuáles son las 3 formas de modificar los perfiles?

1. Comando + Argumento

```latex
\textmd{Text}
```

2. Bloque + Orden

```latex
{
\mdseries
Text
}
```

3. Entornor

```latex
\begin{mdseries}
Text
\end{mdseries}
```
