# Caracteres especiales

## ¿Cúales son los caraceteres especiales?

```latex
    $ & _ ~ ^  # \ { } % 
```

% comentario (compilador omite lo que le sigue)

## ¿Cómo podemos escribir estos caracteres especiales?

Utilizando el backslash `\`

```latex
    \$ \& \_ \# \{ \} \% 
```

Casos especiales `\verb''`

```latex
    \verb'~' \verb'^' \verb'\'  
```
